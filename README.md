## 🚀 Getting Started

### Step 1: Upload Images
You need to add 7 images to the `images/` folder:
1. **1.jpg** - Portrait in blue outfit
2. **2.jpg** - Rooftop cityscape photo
3. **3.jpg** - Portrait in blue (sweet smile)
4. **4.jpg** - Rooftop with yellow flowers
5. **5.jpg** - Red dress with embroidery
6. **6.jpg** - Light green outfit selfie
7. **7.jpg** - Rooftop scenic shot

### Step 2: Upload Music (Optional)
Add `piano.mp3` to the `music/` folder for background music.

### Step 3: Enable GitHub Pages
1. Go to repository **Settings**
2. Scroll to **Pages** section
3. Select `main` branch as source
4. Click **Save**

Your website will be live at: `https://akibaraf170-eng.github.io/anniversary-wish/`
